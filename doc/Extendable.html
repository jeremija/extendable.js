override(p_methodName, p_newMethod) → {Object}

Overrides a method on an existing object with the ability to call the super method. The super parameter (overridden method) will then be added as the first argument to the list. Remember to always add the p_super argument as the first argument when defining a new method and after extending just call the method as you usually would (without the super argument). Also, you can't name the parameteras 'super' because it's a reserved keyword in JavaScript, use 'p_super' or something like that.
Parameters:
